EzBridge API Commands

Real-Time Events [CLIENT-SIDE OK]

Envie e sincronize dados de movimento/interação entre usuários de forma rápida:

socket.emit("RemoteEvent", { type: "seu_tipo", data: "sua_payload" });

User Identification [CLIENT-SIDE OK]

Recupere o ID único permanente do emissor da requisição:

socket.emit("RemoteEvent", { type: "ebuserid_get" }); // O servidor responderá apenas para você com: // { type: "userid_response", userId: "ID_UNICO_DO_CLIENTE" }

EBstorage (Database) [SERVER-SCRIPT ONLY]

Por segurança, comandos de banco de dados não podem mais ser disparados pelos clientes diretamente. Eles devem ser chamados dentro do seu Server Script:

1. Adicionar Dado:

db.add("pasta_chave", "valor_ou_base64", userId);

2. Recuperar Dado:

db.fetch("pasta_chave", userId);

3. Remover Dado:

db.remove("pasta_chave", userId);

4. Limpar Tudo:

db.clear(userId);
