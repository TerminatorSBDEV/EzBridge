EzBridge API Commands

[CLIENT ACCESSIBLE APIs]

Pode ser disparado diretamente pelos navegadores/jogadores:

socket.emit("RemoteEvent", { type: "seu_evento", data: "sua_payload" });
socket.emit("RemoteEvent", { type: "ebuserid_get" });

[SERVERSCRIPT RESTRICTED APIs]

Se um ServerScript estiver ativo, estes comandos de banco de dados são restritos ao script do servidor por segurança:

ebdata.add("ebdata_add_pasta_chave", data, userId);
ebdata.fetch("ebdata_fetch_pasta", userId);
ebdata.clear(userId);
