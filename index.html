How to Connect Your Server

EzBridge API Commands

Real-Time Events (Client-Side Allowed)

Envie pacotes sincronizados de até 250KB:

socket.emit("RemoteEvent", { type: "seu_tipo", data: "sua_payload" });

User Identification (Client-Side Allowed)

Recupere o ID único permanente do emissor da requisição:

socket.emit("RemoteEvent", { type: "ebuserid_get" }); // O servidor responderá apenas para você com: // { type: "userid_response", userId: "ID_UNICO_DO_CLIENTE" }

EBstorage Commands (Database - Server Script Restricted)

Os comandos do banco abaixo exigem execução / aprovação do Server Script:

1. Adicionar/Atualizar Dado:

socket.emit("RemoteEvent", { type: "ebdata_add_pasta_subpasta_chave", data: "valor_ou_base64" });

2. Recuperar Dado:

socket.emit("RemoteEvent", { type: "ebdata_fetch_pasta_subpasta" });

3. Remover Dado:

socket.emit("RemoteEvent", { type: "ebdata_remove_pasta_subpasta_chave" });

4. Criar Pasta de Organização:

socket.emit("RemoteEvent", { type: "ebdata_addfolder", data: { folderName: "sua_pasta" } });

5. Limpar Tudo:

socket.emit("RemoteEvent", { type: "ebdata_clear" });
