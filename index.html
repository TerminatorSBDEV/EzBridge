EBstorage Commands (SERVER-SCRIPT LICENSE ONLY)

Por segurança, o banco físico em disco só pode ser manipulado se o seu script de servidor autorizar:

ebdata_add(caminho, dado); // Grava dado ebdata_fetch(caminho); // Busca dado ebdata_remove(caminho); // Remove dado ebdata_clear(); // Limpa banco do servidor virtual

Server Inter-Connection (SERVER-SCRIPT ONLY)

ebserver_remote_global(type, data); // Dispara evento em rede para todos ebserver_remote_serverid(partnerId, type, data); // Dispara evento para outro servidor virtual ebserver_remote_listen_serverid(partnerId); // Inicia canal persistente bi-direcional
