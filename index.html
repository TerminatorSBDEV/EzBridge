How to Connect Your Server

* For you and a different server to communicate, you both need to fire the "ebserver_remote_listen_serverid" API, make sure to replace serverid with your partner's server id, and he with your server id. You can copy your server id/auth by going on the "How to Connect Your Server" UI and copying the Server Auth Key.

EzBridge API Commands

Real-Time Events (Clientside)

socket.emit("RemoteEvent", { type: "seu_tipo", data: "sua_payload" });

User Identification (Clientside)

socket.emit("RemoteEvent", { type: "ebuserid_get" });

Important DB APIs (Requires Serverscript Permission)

* Estas APIs só podem ser disparadas se o seu script de servidor autorizar:

socket.emit("RemoteEvent", { type: "ebdata_add_chave", data: "valor" }); socket.emit("RemoteEvent", { type: "ebdata_fetch_chave" }); socket.emit("RemoteEvent", { type: "ebdata_remove_chave" });
