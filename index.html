ServerScript-Only Protected APIs (Confidential)

As chamadas diretas abaixo são bloqueadas nos navegadores por segurança. Elas podem ser executadas apenas de forma privada por scripts no lado do servidor:

// Escreve ou gerencia o banco de dados físico persistente (EBstorage.js) db.add("players_Guest_coins", 50, userId); db.fetch("players_Guest_coins", userId); // Comunicação Cross-Server nativa segura: ebserver_remote_global("broadcast_type", "data"); ebserver_remote_serverid("PARTNER_SERVER_ID", "com_type", "payload"); // Libera outro servidor para conversar com o seu: ebserver_remote_listen_serverid("PARTNER_SERVER_ID"); // Nota: para se comunicarem, ambos precisam se escutar mutuamente.
