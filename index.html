EzBridge API Commands

Real-Time Events (Fired by Client)

Permitido ser emitido nativamente por qualquer cliente:

socket.emit("RemoteEvent", { type: "seu_tipo", data: "sua_payload" });
socket.emit("RemoteEvent", { type: "ebuserid_get" }); // Retorna seu ID único

Privileged Events (ServerScript Only)

Bloqueado no cliente se um script estiver ativo. Executado dentro do sandbox:

1. Inserir/Atualizar Dado (ebdata_add):

db.add("pasta_chave", "valor", "id_usuario");

2. Recuperar Dado (ebdata_fetch):

db.fetch("pasta_chave", "id_usuario");

3. Remover Dado (ebdata_remove):

db.remove("pasta_chave", "id_usuario");

4. Criar Pasta Organizadora:

db.addFolder("sua_pasta", "id_usuario");
