EzBridge API Commands

Real-Time Events (Client-Side Available)

Envie pacotes sincronizados livremente entre clientes:

socket.emit("RemoteEvent", { type: "seu_tipo", data: "sua_payload" });

User Identification (Client-Side Available)

Recupere a ID única do emissor da requisição:

socket.emit("RemoteEvent", { type: "ebuserid_get" }); // Resposta retornada: { type: "userid_response", userId: "ID_DO_USER" }

EBstorage Database (Requires Server Script)

* Bloqueado por padrão para envio direto de clientes. Deve ser executado a partir do seu código no servidor virtual (Sandbox):

1. Adicionar Dado:

db.add("pasta_subpasta_chave", "valor_ou_base64");

2. Recuperar Dado:

db.fetch("pasta_subpasta");

3. Remover Dado:

db.remove("pasta_subpasta_chave");

4. Limpar Tudo:

db.clear();
