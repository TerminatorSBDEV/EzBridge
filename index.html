How to Connect Your Server

EzBridge API & Commands

CLIENT-SIDE API (Safe, can be executed directly)

Permite enviar pacotes e sincronizar sem restrições:

socket.emit("RemoteEvent", { type: "seu_tipo", data: "sua_payload" });

Gera o ID do usuário de forma permanente:

socket.emit("RemoteEvent", { type: "ebuserid_get" });

SERVER-SIDE EXCLUSIVE API (Requires ServerScript permission)

Comandos do banco de dados (EBstorage) são interceptados e autorizados apenas por scripts:

1. Adicionar/Atualizar Dado (ebdata_add):

socket.emit("RemoteEvent", { type: "ebdata_add_score", data: 120 });

2. Recuperar Dado (ebdata_fetch):

socket.emit("RemoteEvent", { type: "ebdata_fetch_score" });
