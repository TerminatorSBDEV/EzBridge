EzBridge API Reference

Client-Safe APIs (No Script Permission Required)

Qualquer soquete conectado de forma anônima pode disparar estes eventos:

// Envia pacotes brutos sincronizados para outros usuários: socket.emit("RemoteEvent", { type: "seu_evento", data: "dados" }); // Recupera o ID único permanente do remetente: socket.emit("RemoteEvent", { type: "ebuserid_get" }); // Retorna: { type: "userid_response", userId: "ID_UNICO" }

Server-Only APIs (Protected Sandbox Required)

Estes comandos são bloqueados para clientes! Só podem ser chamados de dentro do seu script de servidor:

// Modifica a base física EBstorage.js (3GB limite): db.add("pasta_chave", "valor"); db.remove("pasta_chave"); db.clear(); // Limpa banco // Recupera dado de persistência física: let dado = db.fetch("pasta_chave"); // Escuta eventos dos clientes de forma sincronizada: onEvent("tipo_evento", (clientId, data) => { console.log("Recebido do cliente:", clientId, data); // Envia resposta para a sala virtual inteira: io.emit("resposta_servidor", "dados"); });
