EBstorage & Client Broker commands

These commands can be triggered via Socket events on the client side:

// 1. Storage Operations (EBstorage.js) socket.emit('ebstorage_action', { action: 'ebdata_add', path: 'coins', value: 50 }); socket.emit('ebstorage_action', { action: 'ebdata_fetch', path: 'coins' }); socket.on('db_response', (data) => { console.log(data.value); }); // 2. Folder Creation socket.emit('ebstorage_action', { action: 'ebdata_addfolder', path: 'users/Guest' }); // 3. Clear Storage socket.emit('ebstorage_action', { action: 'ebdata_clear' });
